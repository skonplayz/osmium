# How to use AI to generate Osmium code

## 1. Head to the [Osmium website](https://osmium-playground.vercel.app) and scroll down to your version in the "Releases" section.
<img width="982" height="791" alt="image" src="https://github.com/user-attachments/assets/00230690-a887-46ad-8c47-377b0608172a" />

---


## 2. Click on the "API" button. It should have an opened eye emoji. 
<img width="81" height="39" alt="api button" src="https://github.com/user-attachments/assets/33fc175e-7553-4d36-8105-9d5023845d27" />

---


## 3. Copy the API with the Copy button in the top right of the popup.
<img width="912" height="854" alt="image" src="https://github.com/user-attachments/assets/fc025f81-836f-4595-afde-c335218edab5" />

---


## 4. Ask your AI of choice to generate any script you like! Paste the API you have copied. 
### NOTE: the api is longer than 2,000 lines long, and most AIs block this length. Try sending it as a markdown file.
<img width="832" height="721" alt="image" src="https://github.com/user-attachments/assets/ed698da6-d546-4905-a6d4-8baf3d03299c" />

It's as easy as that!
---

# I heavily advise AGAINST using AI to generate Osmium code. Osmium is an incredibly simple language, but if you are having any issues and you want to double-check feel free to use AI.
